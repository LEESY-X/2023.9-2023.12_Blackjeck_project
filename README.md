# CLI-based 블랙잭 개발 프로젝트

## **Project Summary**

### **프로젝트 개요**

**주제:** 파이썬 기반 콘솔형 블랙잭(Blackjack) 게임 개발.

**프로젝트 목표:** 객체지향 프로그래밍(OOP) 개념을 활용하여 딜러와 플레이어 간의 카드 대결 로직을 구현하고, 베팅 및 유저 데이터 관리 시스템을 구축하고자 하였다.

**사용 데이터:** 유저 정보 및 전적 데이터 (.txt 파일 기반 로컬 DB).

### **주요 기술**

**언어 및 환경:** Python, VS Code.

**프로그래밍 기법**

- OOP (객체 지향): Card, Deck, Player, Dealer, Game 등 기능별 클래스 모듈화 설계.
- File I/O: 텍스트 파일을 활용한 유저 데이터(칩, 승률, 전적) 영구 저장 및 불러오기 구현.
- Algorithm: 카드 셔플(Shuffle), 딜링, 점수 계산(A를 1 또는 11로 처리) 로직 구현.

### **구현 기능 및 성과**

**게임 로직:**

- **기본 규칙:** 카드 합이 21에 가까운 쪽이 승리하며, 21을 초과하면 패배(Burst).
- **액션 구현:** 플레이어는 상황에 따라 카드 추가(Hit) 또는 멈춤(Stay) 선택 가능.
- **딜러 규칙:** 딜러는 카드 합이 16 이하일 경우 의무적으로 카드를 추가하는 카지노 룰 적용.

**베팅 시스템:** 게임 시작 전 칩을 베팅하고, 승리 시 배당 지급, 무승부(Push) 시 반환 로직 구현.

**유저 관리:** 사용자 이름 입력 시 기존 데이터를 로드하거나 신규 생성하며, 게임 종료 후 변동된 칩과 전적을 자동 저장하여 지속적인 플레이 지원.
